MernAuth & CRUDify

Your project is a web application built using the MERN stack (MongoDB, Express.js, React, and Node.js) that facilitates user authentication (login and signup) and incorporates CRUD (Create, Read, Update, Delete) operations.

Project Summary...



Your project features a login and signup authentication system that enables users to securely create accounts and access their profiles.

User Registration: Users can sign up by providing their details, which are validated and securely stored in a MongoDB database.

Password Encryption: User passwords are hashed before storage to ensure security and protect against unauthorized access.

Login Functionality: Users can log in using their credentials, with checks to authenticate their identity.

Session Management: User sessions are managed to keep users logged in, enhancing the user experience while maintaining security.

User can also Logout comfortably whenever they want


Key Features
User-specific item management with email-based authentication

Real-time CRUD operations without page refresh

Interactive form handling for item creation and updates

Responsive table display of user items

Date and time tracking for each item

Confirmation dialogs for delete operations

Session management with logout functionality
