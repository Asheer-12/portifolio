TechHub - E-Commerce Website

TechHub is a modern e-commerce website created using HTML and CSS, designed to showcase and sell the latest gadgets. It offers a clean and intuitive layout with interactive features, ensuring a seamless user experience for browsing and purchasing tech products.

Project Summary...



Homepage Navigation: When users land on the homepage, they are greeted with a hero section featuring a tagline and a "Shop Now" button. Action: Clicking the "Shop Now" button scrolls them to the Featured Products Section, where they can browse the highlighted items. Category Navigation (Navbar):

The navigation bar at the top allows users to explore specific product categories: All: Displays all products available on the website. Phones: Filters and shows products related to mobile phones. Laptops: Displays only laptops and similar products. Watches: Focuses on wearable devices like smartwatches. Audio: Lists audio-related gadgets such as headphones and speakers. Action: Users can click on any category link to filter the products displayed in the grid.


Search Functionality: A search icon is provided in the navigation bar for users to quickly look up specific products. Action: Clicking the search icon opens a search bar where users can enter a keyword (e.g., "headphones") to find relevant items.
